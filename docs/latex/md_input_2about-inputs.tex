The {\ttfamily input} folder contains all the input files required for the Antenna Mapping project. These files include the initial matrix configurations.\hypertarget{md_input_2about-inputs_autotoc_md19}{}\doxysection{\texorpdfstring{File Structure}{File Structure}}\label{md_input_2about-inputs_autotoc_md19}

\begin{DoxyItemize}
\item {\ttfamily input/}\+: Directory containing all input files.
\end{DoxyItemize}\hypertarget{md_input_2about-inputs_autotoc_md20}{}\doxysection{\texorpdfstring{Input File Format}{Input File Format}}\label{md_input_2about-inputs_autotoc_md20}
Each input file should follow a specific format to be correctly processed by the program\+:


\begin{DoxyItemize}
\item The file should contain a matrix where dots ({\ttfamily .}) represent empty spaces and characters represent antennas.
\item Each row of the matrix should be on a new line.
\item The matrix should be rectangular, meaning all rows should have the same number of columns.
\end{DoxyItemize}\hypertarget{md_input_2about-inputs_autotoc_md21}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{md_input_2about-inputs_autotoc_md21}
