\doxysubsection{Detailed Description}
Console front-\/end implementation for the antenna effect program. 

This file contains the implementation of user interface functions for interacting with the user via the console. It includes functions to display menus, read user input, and print graph vertices. All stdout/stderr interaction is handled here to keep core logic pure and testable.

\begin{DoxyNote}{Note}
Language strings are embedded at compile time via {\ttfamily \doxylink{strings_8h}{strings.\+h}}. 
\end{DoxyNote}
