The {\ttfamily output} folder contains all the output files generated by the Antenna Mapping project. These files include the saved antenna configurations and the calculated effect positions.\hypertarget{md_output_2about-outputs_autotoc_md25}{}\doxysection{\texorpdfstring{File Structure}{File Structure}}\label{md_output_2about-outputs_autotoc_md25}

\begin{DoxyItemize}
\item {\ttfamily output/}\+: Directory containing all output files.
\end{DoxyItemize}\hypertarget{md_output_2about-outputs_autotoc_md26}{}\doxysection{\texorpdfstring{Output File Format}{Output File Format}}\label{md_output_2about-outputs_autotoc_md26}
Each output file follows a specific format to ensure consistency and ease of processing\+:


\begin{DoxyItemize}
\item The file contains a matrix where dots ({\ttfamily .}) represent empty spaces and characters represent antennas.
\item Each row of the matrix is on a new line.
\item The matrix is rectangular, meaning all rows have the same number of columns.
\end{DoxyItemize}\hypertarget{md_output_2about-outputs_autotoc_md27}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{md_output_2about-outputs_autotoc_md27}
