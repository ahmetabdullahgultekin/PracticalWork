The {\ttfamily src} folder contains all the source code files for the Antenna Mapping project. These files implement the core functionalities of the project, including reading input files, processing antenna data, calculating effect positions, and handling user interactions through the menu.\hypertarget{md_src_2about-source-codes_autotoc_md42}{}\doxysection{\texorpdfstring{File Structure}{File Structure}}\label{md_src_2about-source-codes_autotoc_md42}

\item {\ttfamily src/}\+: Directory containing all source code files.
\begin{DoxyItemize}
\item {\ttfamily \doxylink{main_8c}{main.\+c}}\+: The main entry point of the program.
\item {\ttfamily \doxylink{menu_8c}{menu.\+c}}\+: Implementation of menu-\/related functions.
\item {\ttfamily \doxylink{io__ops_8c}{io\+\_\+ops.\+c}}\+: Implementation of input/output operations, including file reading and writing.
