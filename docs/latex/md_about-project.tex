\hypertarget{md_about-project_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md_about-project_autotoc_md1}
This project is a practical work for the Advanced Data Structures course. It involves reading a matrix from a file where dots represent empty spaces and characters represent antennas. The program calculates the effect positions of the antennas using specific formulas and provides functionalities to insert, remove, print antennas, and clear the lists.\hypertarget{md_about-project_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_about-project_autotoc_md2}

\begin{DoxyItemize}
\item Load antennas from an input file
\item Print the current antenna map
\item Print the effect positions of antennas
\item Insert a new antenna
\item Remove an existing antenna
\item Save antennas to an output file
\item Save effect positions to an output file
\end{DoxyItemize}\hypertarget{md_about-project_autotoc_md3}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md_about-project_autotoc_md3}
