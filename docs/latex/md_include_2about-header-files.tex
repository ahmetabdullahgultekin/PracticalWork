The {\ttfamily include} folder contains all the header files for the Antenna Mapping project. These files declare the functions and data structures used throughout the project, providing the necessary interfaces for the source code files.\hypertarget{md_include_2about-header-files_autotoc_md21}{}\doxysection{\texorpdfstring{File Structure}{File Structure}}\label{md_include_2about-header-files_autotoc_md21}
