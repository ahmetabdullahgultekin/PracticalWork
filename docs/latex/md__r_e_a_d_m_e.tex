\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{md__r_e_a_d_m_e_autotoc_md1}
This project is a practical work for the Advanced Data Structures course (ESI). It involves reading a matrix from a file where dots represent empty spaces, and characters represent antennas. The program calculates the effect positions of antennas using specific formulas and provides functionalities to insert, remove, print antennas, and clear the lists.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item Load antennas from an input file.
\item Print the current antenna map.
\item Print the effect positions of antennas.
\item Insert a new antenna.
\item Remove an existing antenna.
\item Save antennas to an output file.
\item Save effect positions to an output file.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Effect Position Calculation}{Effect Position Calculation}}\label{md__r_e_a_d_m_e_autotoc_md3}
Effect positions are calculated using the formulas\+:


\begin{DoxyItemize}
\item {\ttfamily L1 = (2 \texorpdfstring{$\ast$}{*} row1 -\/ row2, 2 \texorpdfstring{$\ast$}{*} col1 -\/ col2)}
\item {\ttfamily L2 = (2 \texorpdfstring{$\ast$}{*} row2 -\/ row1, 2 \texorpdfstring{$\ast$}{*} col2 -\/ col1)}
