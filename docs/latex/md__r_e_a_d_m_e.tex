This project implements the solution for the “\+Advanced Data Structures” (EDA) course practical assignment at IPCA. ~\newline
 It models a city’s antenna network, analyzes dangerous electromagnetic interference zones according to specified rules, and provides both linked-\/list and graph-\/based analysis. The solution is modular, testable, and fully documented with Doxygen.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Loads antenna maps}} from files of arbitrary size and content
\item {\bfseries{Dynamic linked list}} management for antenna information and dangerous points
\item {\bfseries{Graph-\/based modeling}} for advanced operations (DFS, BFS, path finding)
\item {\bfseries{Dangerous point calculation}} following project rules (frequency alignment and distance)
\item {\bfseries{Finds intersections}} between dangerous zones generated by different frequencies
\item {\bfseries{Interactive CLI menu}} for all core operations
\item {\bfseries{Modular source code}} with clean separation of logic and UI
\item {\bfseries{Doxygen documentation}} for all modules and functions
\item {\bfseries{Test scenarios}} and sample input files included
